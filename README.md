# Técnicas de Programación – Ejemplos Prácticos

## Descripción

Este repositorio contiene ejemplos prácticos de las principales **técnicas de programación orientada a objetos**, desarrollados como parte de la asignatura **Técnicas de Programación**. El objetivo es aplicar de forma correcta los conceptos vistos en clase mediante código claro, organizado y documentado.

Las técnicas abordadas en este repositorio son:

* Abstracción
* Encapsulación
* Herencia
* Polimorfismo

---

## Estructura del Repositorio

La organización del repositorio se divide en carpetas, una por cada técnica de programación:

```

```

Cada carpeta contiene ejemplos de código y comentarios que explican el funcionamiento y propósito de cada técnica.

## Detalle de las Técnicas

### 1. Abstracción

Se utiliza para representar las características esenciales de un objeto, ocultando los detalles internos. En el ejemplo se emplean clases abstractas y métodos abstractos.

### 2. Encapsulación

Permite proteger los datos de una clase mediante modificadores de acceso, asegurando que los atributos solo puedan ser modificados a través de métodos controlados.

### 3. Herencia

Facilita la reutilización de código, permitiendo que una clase hija herede atributos y métodos de una clase padre.

### 4. Polimorfismo

Hace posible que un mismo método se comporte de diferentes maneras según el objeto que lo invoque, mejorando la flexibilidad del programa.
